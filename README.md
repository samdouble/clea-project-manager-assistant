# project-manager-assistant (Cléa)

Optionally, create a `.env` file like this:
```
ANTHROPIC_API_KEY=
LINEAR_API_KEY=
```
If you don't, you are going to be asked the informations through the CLI.

Run:
```
poetry run clea
```

Ask questions:
```
Hi Cléa, I'd like to see my issues for next cycle
```
